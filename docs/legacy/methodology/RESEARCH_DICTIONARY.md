# Purpose

This document defines the official research terminology used throughout
the College Football ATS Research Platform. These definitions provide a
common vocabulary for documentation, software, datasets, and statistical
studies.

------------------------------------------------------------------------

# Core Research Terms

## ATS (Against the Spread)

The result of a game relative to the published betting spread rather
than the straight-up winner.

## ATS Margin

The official calculated margin:

    (home_score - away_score) + spread

Positive values indicate a home cover, negative values indicate an away
cover, and zero represents a push.

## Cover

A team performs better than the betting spread predicts.

## Push

A game in which neither team covers because the ATS Margin equals zero.

## Lined Game

A game with a documented betting spread available for analysis.

## Effective Rank

The official ranking selected using the priority:

1.  CFP
2.  AP
3.  Coaches

## Research Universe

Default research population:

-   FBS only
-   Regular season
-   Minimum 75 lined games for leaderboard studies

# Statistical Terms

## Sample Size

The number of qualifying games included in a study.

## Minimum Sample

Default publication threshold:

-   75 lined games

## Outlier

An observation sufficiently different from the remaining sample to
warrant additional investigation.

------------------------------------------------------------------------

# Research Principles

All research should:

-   Be reproducible.
-   Document assumptions.
-   Distinguish observation from interpretation.
-   Report sample sizes.
-   Avoid overstating statistical significance.
